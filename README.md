## Como rodar a aplicação
Você precisa ter instalado globalmente as seguintes aplicações
> Node
> NPM 
> Typescript
> Nodemon


### Você precisará criar um arquivo .env com os seguintes dados
```

MYSQL_DB = sysvendas
MYSQL_USER = 
MYSQL_PASSWD =

```
